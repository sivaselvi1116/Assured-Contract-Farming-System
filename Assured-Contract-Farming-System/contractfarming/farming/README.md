# 🌾 Assured Contract Farming System for Stable Market Access

## 📌 Project Overview

The Assured Contract Farming System for Stable Market Access is a web-based application developed using Django to establish a transparent and reliable platform connecting farmers, buyers, and administrators. The system simplifies the contract farming process by enabling secure registration, contract management, and efficient data handling while improving market access for farmers.

---

## 🎯 Project Objectives

- To provide a secure platform for contract farming.
- To simplify farmer and buyer registration.
- To improve transparency between farmers and buyers.
- To maintain accurate contract records.
- To reduce manual paperwork through digital management.

---

## ✨ Key Features

- Secure Admin Login
- Farmer Registration
- Buyer Registration
- Contract Registration
- Registration Success Confirmation
- Database Management using SQLite
- User-Friendly Web Interface

---

## 🏗️ Project Modules

### 1. Home Module
Displays the project homepage and navigation.

### 2. Admin Module
Allows the administrator to manage registrations and monitor the system.

### 3. Farmer Module
Enables farmers to register and provide their farming details.

### 4. Buyer Module
Allows buyers to register and participate in contract farming.

### 5. Contract Module
Stores and manages farming contract information between farmers and buyers.

### 6. Success Module
Displays a confirmation page after successful registration.

## 🚀 Installation and Execution

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Install Django:


pip install django


4. Run the server:


python manage.py runserver


5. Open your browser and visit:


http://127.0.0.1:8000/
